# 👁️ 시각 장애인들을 위한 점자리딩 및 음성 출력기
> **Braille Reading & Audio Output Program using YOLOv5**
> 2024년 시스템반도체공학과 캡스톤 디자인 (Capstone Design)

## 📌 프로젝트 개요 (Overview)
본 프로젝트는 점자 문해율이 낮은 시각장애인들을 위해 **객체 인식 기술(AI)**과 **음성 합성 기술(TTS)**을 결합한 보조 공학 솔루션입니다. 카메라로 점자를 촬영하면 실시간으로 해당 내용을 분석하여 한국어 음성으로 안내합니다.

## 🛠 기술 스택 (Tech Stack)
### **AI & Software**
- **Object Detection:** `YOLOv5` (점자 위치 및 문자 객체 인식)
- **Language:** `Python`
- **Libraries:** - `gTTS (Google Text-To-Speech)`: 텍스트 음성 변환
  - `Playsound`: 오디오 시스템 제어 및 재생
  - `OpenCV`: 이미지 전처리 및 결과 바운딩 박스 시각화

### **Environment**
- **Dataset:** 실생활 점자 데이터 (지하철역 등 5곳 수집)
- **Target:** 실시간 캠 라이브 인식 및 이미지 분석

---

## 🏗 주요 연구 및 구현 내용

### 1. YOLOv5 기반 객체 인식 (Object Detection)
- **모델 선정:** 실시간 추론 속도가 빠르고 정확도가 높은 **YOLOv5** 모델을 활용하여 점자판 내의 개별 점자 유닛을 인식합니다.
- **인식 로직:** 카메라 입력 영상에서 점자 객체를 탐지하고, 인식된 영역의 신뢰도(Confidence Score)를 바탕으로 텍스트 추출을 진행합니다.

### 2. 점자 분석 및 디코딩 (Decoding)
- **점자 추론:** 바운딩 박스로 지정된 영역을 3x2 매트릭스로 세분화하여 분석합니다.
- **이진화 처리:** 하얀 지역의 외접원 면적 계산 등 이미지 처리를 통해 점자의 패턴을 정확히 판별합니다.
- **매핑:** 판별된 점자 패턴을 규격에 맞는 한글/숫자 데이터로 변환합니다.

### 3. 음성 출력 시스템 (Audio Output)
- 변환된 텍스트 데이터를 `gTTS`를 통해 음성 데이터로 합성합니다.
- 사용자가 실시간으로 정보를 확인할 수 있도록 즉각적인 오디오 피드백을 제공합니다.

## 📊 연구 결과 (Performance)
- **인식 모델:** YOLOv5 기반 점자 인식 알고리즘
- **테스트 데이터:** 실물 점자 단어 30개 (지하철역 등 실생활 환경)
- **최종 지표:**
  - **캠 라이브 안정도:** **85%**
  - **정확도 달성:** 점자 인식 성공률 90% 이상 시 안정적 음성 출력 지원
