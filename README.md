## 📌 프로젝트 개요 (Overview)
본 프로젝트는 점자 문해율이 낮은 시각장애인들의 정보 접근성을 개선하기 위해 개발되었습니다. **YOLOv5 기반 객체 인식** 기술을 통해 실생활 속 점자를 실시간으로 탐지하고, 분석된 내용을 한국어 음성으로 변환하여 안내하는 보조 공학 솔루션입니다.



## 🛠 기술 스택 (Tech Stack)
### **AI & Software**
- **Object Detection:** `YOLOv5` (점자 탐지 및 위치 추론)
- **Language:** `Python`
- **Libraries:** - `gTTS (Google Text-To-Speech)`: 텍스트-음성 합성
  - `Playsound`: 시스템 오디오 출력 제어
  - `OpenCV`: 이미지 전처리 및 결과 시각화

---

## 🏗 주요 구현 내용

### 1. YOLOv5 기반 점자 인식
- 실시간 객체 인식 모델인 **YOLOv5**를 활용하여 점자 유닛을 정확하게 탐지합니다.
- 고화질 카메라 환경에서 **mAP 0.533**의 성능을 확보하여 신뢰도 높은 데이터를 추출합니다.

### 2. 점자 디코딩 알고리즘
- 탐지된 객체를 **3x2 매트릭스**로 분할하여 점자의 돌출 패턴을 분석합니다.
- **이미지 이진화(Binarization)** 및 외접원 면적 계산 알고리즘을 통해 점자 패턴을 한글/숫자 데이터로 매핑합니다.



### 3. 실시간 음성 안내 시스템
- 분석된 텍스트를 `gTTS`를 통해 한국어 음성으로 합성하고, 사용자가 즉각 정보를 파악할 수 있도록 오디오 피드백을 제공합니다.

## 📊 연구 결과 및 성능 지표 (Performance)
*지하철역 등 5곳의 실생활 점자 데이터(단어 30개)를 기반으로 테스트를 진행했습니다.*

* **안정도 측정 기준:** 점자 인식 성공률 **90% 이상** 도달 시 '안정' 판정
* **테스트 이미지 안정도:** **85%** (정적 이미지 환경)
* **캠 라이브 안정도:** **45%** (실제 환경 촬영)
  * *분석:* 실시간 영상의 조도 및 흔들림 등 외부 요인에 따른 성능 저하를 확인하였으며, 이를 향후 알고리즘 고도화 과제로 설정함.
